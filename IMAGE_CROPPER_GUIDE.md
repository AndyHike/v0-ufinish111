# Image Cropper & Editor Guide

## Нові можливості

Тепер при завантаженні фото на статтю доступні наступні функції:

### 1. **Обрізання та центрування**
- Після вибору файлу автоматично відкривається редактор зображення
- Перетягуйте зображення для центрування
- Регулюйте масштаб (zoom)
- Поворот на 90 градусів

### 2. **16:9 Aspect Ratio**
- Всі зображення статей обрізаються в формат 16:9
- Це стандартний формат для веб-сайтів та соціальних мереж
- Забезпечує консистентний вигляд у списках статей

### 3. **Автоматична обробка**
- Після обрізання зображення:
  - Конвертується в WebP (75% менше розміру)
  - Масштабується до 1200px ширини
  - Оптимізується для швидкого завантаження

## Як використовувати

### Додавання нового зображення

1. **Натисніть на поле для завантаження** або перетягніть файл
2. **Редактор відкривається автоматично**:
   - Перетягуйте зображення для центрування
   - Прокручуйте мишкою або використовуйте повзунок для масштабування
   - Натисніть "Rotate 90°" для повороту
3. **Натисніть "Apply Crop"** для підтвердження
4. Зображення завантажується на Cloudflare S3

### Заміна існуючого зображення

- Натисніть на кнопку редагування (олівець) на попередній версії
- Виберіть нове зображення
- Виконайте кроки редагування

### Видалення зображення

- Натисніть червону кнопку X на попередній версії

## Контролі в редакторі

| Контроль | Дія |
|----------|-----|
| **Drag & Drop** | Переміщення та центрування зображення |
| **Scroll / Slider** | Масштабування (від 1x до 3x) |
| **Rotate Button** | Поворот на 90 градусів |
| **Apply Crop** | Підтвердження та завантаження |
| **Cancel** | Скасування без змін |

## Особливості

- Інтерактивна сітка для вирівнювання
- Режим темного фону для кращої видимості
- Лайв-превю обрізаного результату
- Швидко і без втрат якості

## Технічні деталі

### Бібліотека
- **react-easy-crop**: Потужна та легка бібліотека для обрізання
- Безпечна обробка на клієнті (без навантаження на сервер)

### Формат
- **Aspect Ratio**: 16:9 (фіксований)
- **Output Format**: WebP (JPEG fallback)
- **Compression**: 90% якість
- **Max Width**: 1200px

### Файли

- `/components/admin/image-cropper.tsx` - Редактор зображень
- `/components/admin/image-upload-field.tsx` - Поле завантаження з інтеграцією редактора
