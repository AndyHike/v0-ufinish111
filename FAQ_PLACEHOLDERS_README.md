# FAQ Плейсхолдери для Послуг

Ця система дозволяє адміністраторам створювати універсальні FAQ для послуг з автоматичною персоналізацією для кожної моделі та бренду.

## Як це працює

### 1. В адміні при редагуванні FAQ

Коли ви редагуєте FAQ послуги, ви можете використовувати спеціальні плейсхолдери в текстах питань та відповідей:

```
Питання: {{brand}} {{model}} не включається - як це виправити?
Відповідь: {{service}} для {{device}} займає {{duration_hours}} годину(н). 
Гарантія {{warranty_months}} {{warranty_period}}.
Ціна: {{price}}
```

### 2. На сайті

Коли користувач переглядає сторінку послуги, всі плейсхолдери автоматично замінюються на реальні дані:

**URL:** `/cs/services/screen-replacement?model=iphone-11-pro-max`

**Результат:**
```
Питання: Apple iPhone 11 Pro Max не включається - як це виправити?
Відповідь: Screen Replacement для Apple iPhone 11 Pro Max займає 1 годину. 
Гарантія 6 months.
Ціна: 1,490 CZK
```

## Доступні плейсхолдери

| Плейсхолдер | Опис | Приклад | Коли доступний |
|--------------|------|---------|---|
| `{{model}}` | Назва моделі | iPhone 11 Pro Max | Коли вибрана модель |
| `{{model_slug}}` | URL модель | iphone-11-pro-max | Коли вибрана модель |
| `{{brand}}` | Назва бренду | Apple | Коли вибрана модель |
| `{{brand_slug}}` | URL бренду | apple | Коли вибрана модель |
| `{{service}}` | Назва послуги | Screen Replacement | Завжди |
| `{{service_slug}}` | URL послуги | screen-replacement | Завжди |
| `{{warranty_months}}` | Місяці гарантії | 6 | Коли вказано |
| `{{warranty_period}}` | Період гарантії | months / days | Коли вказано |
| `{{duration_hours}}` | Тривалість в годинах | 1, 2, 3 | Коли вказано |
| `{{price}}` | Ціна послуги | 1,490 CZK | Коли відома |
| `{{device}}` | Повна назва пристрою | Apple iPhone 11 Pro Max | Коли вибрана модель |

## Як використовувати в адміні

### Крок 1: Відкрити редагування FAQ

1. Перейдіть до адміну → Послуги
2. Виберіть послугу
3. Натисніть "Редагувати FAQ" або "Додати FAQ"

### Крок 2: Додати плейсхолдери

У полях "Питання" та "Відповідь" натисніть кнопку **"Плейсхолдери"**:

![Placeholder Button](https://example.com/placeholder-button.png)

### Крок 3: Вибрати плейсхолдер

З'явиться діалог зі списком усіх доступних плейсхолдерів:
- Виберіть потрібний
- Натисніть **"Вставити"**
- Плейсхолдер буде вставлено в поточну позицію курсора

### Крок 4: Зберегти FAQ

Натисніть **"Зберегти"** внизу форми.

## Приклади FAQ

### Приклад 1: Просте питання

```
Q: Скільки часу займає {{service}}?
A: {{service}} займає {{duration_hours}} годину(н).
```

**Результат для iPhone 11 Pro Max:**
```
Q: Скільки часу займає Screen Replacement?
A: Screen Replacement займає 1 годину.
```

### Приклад 2: Складне питання

```
Q: {{brand}} {{model}} вимагає заміни {{service}} - чи це дорого?
A: {{service}} для {{device}} коштує {{price}} з гарантією {{warranty_months}} {{warranty_period}}.
```

**Результат:**
```
Q: Apple iPhone 11 Pro Max вимагає заміни Screen Replacement - чи це дорого?
A: Screen Replacement для Apple iPhone 11 Pro Max коштує 1,490 CZK з гарантією 6 months.
```

### Приклад 3: FAQ без моделі

```
Q: Яка гарантія на {{service}}?
A: На {{service}} гарантія {{warranty_months}} {{warranty_period}}.
```

Це FAQ буде показано всім користувачам незалежно від моделі, яка вибрана:
```
Q: Яка гарантія на Screen Replacement?
A: На Screen Replacement гарантія 6 months.
```

## Корисні поради

1. **Тестуйте FAQ на різних моделях**
   - Переходьте на сторінку послуги з різними моделями
   - Переконайтеся, що всі плейсхолдери правильно замінюються

2. **Використовуйте {{device}} замість {{brand}} {{model}}**
   - `{{device}}` = "Apple iPhone 11 Pro Max"
   - `{{brand}} {{model}}` = "Apple iPhone 11 Pro Max" (те ж саме, але з двома плейсхолдерами)

3. **Майте на увазі контекст мови**
   - Плейсхолдери замінюються однаково для всіх мов (cs, en, uk)
   - Адаптуйте текст FAQ для кожної мови окремо

4. **Не забувайте про FAQ без моделі**
   - FAQ показується і коли користувач переглядає послугу без вибору моделі
   - Упевніться, що текст робить сенс без плейсхолдерів моделі

## FAQ про FAQ плейсхолдери

**P: Що станеться, якщо плейсхолдер не має значення?**
A: Плейсхолдер залишиться як є (напр. `{{model}}`). Позначте ці місця в FAQ для перевірки.

**P: Можна ли комбінувати плейсхолдери?**
A: Так! Наприклад: `{{brand}} {{model}} {{service}}`

**P: Чи замінюються плейсхолдери в заголовку сторінки?**
A: Ні, тільки в FAQ. Заголовки генеруються окремо на server-side.

**P: Як видалити плейсхолдер?**
A: Просто видаліть текст плейсхолдера вручну з поля. Або відредагуйте FAQ.
